# Cloudflare Manager

Cloudflare Manager is a web application designed to simplify managing multiple URLs compared to the standard Cloudflare site. Built with Flask, it offers an intuitive interface for URL-related tasks.

## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/Sum1Solutions/Cloudflare-Manager.git
   ```

2. **Navigate to the Directory**:

   ```bash
    cd Cloudflare-Manager
    ```

3. **Install Required Packages**:

   ```bash
   pip install -r requirements.txt
   ```

4. **Set Environment Variables**:
Rename `.env.example` to `.env` and fill in the necessary details.

5. **Run the Application**:

   ```bash
   flask run
   ```

## Usage

Visit `http://127.0.0.1:5000/` in your browser to access the Cloudflare Manager dashboard. (Or click the link in the terminal after running `flask run`.)
